Authentication and Session Flow
%%{init: { "theme": "base", "themeVariables": { "primaryColor": "#f0fdf4", "primaryBorderColor": "#22c55e", "lineColor": "#64748b", "fontFamily": "Segoe UI, system-ui, sans-serif", "fontSize": "13px" } }}%% flowchart LR A([User visits any page]) --> B{Session cookie\npresent?} B -- No --> C[middleware: createMiddlewareClient\ngetSession → no session] B -- Yes --> D[middleware: createMiddlewareClient\ngetSession → refreshes JWT] C --> E[Page renders as anonymous\nNo edit UI shown] D --> F[Server component reads\nrefreshed cookie] F --> G{Need user role?} G -- Yes\nAPI route --> H[createRouteHandlerClient\ngetUser → check user_profiles.role] G -- Yes\nClient component --> I[useUserRole hook\nonAuthStateChange listener\ncheck user_profiles.role] G -- No --> J[Render public content] H --> K{Role sufficient?} K -- No --> L[Return 401 / 403] K -- Yes --> M[Execute operation] I --> N[Show/hide UI elements\nbased on role flags]
